# 📄 Criador de Recibos Online

Um sistema web simples e eficiente para criação de recibos personalizados, com geração rápida e opção de exportação em PDF.

---

## 🚀 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de facilitar a criação de recibos digitais de forma prática. O usuário pode preencher os dados diretamente na interface e gerar um recibo pronto para uso, sem necessidade de sistemas complexos.

A aplicação possui edição em tempo real, layout responsivo e foco em usabilidade, sendo ideal para pequenos negócios, vendedores e prestadores de serviço.

---

## ✨ Funcionalidades

- 📝 Preenchimento dinâmico dos dados  
- ✏️ Edição direta (sem botão de editar)  
- 📄 Geração automática de recibo  
- 📥 Exportação em PDF  
- 📱 Layout responsivo (mobile e desktop)  
- ✍️ Campo para assinatura  

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript
